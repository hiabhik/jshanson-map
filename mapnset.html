Q."Problem: You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps) Input: abcadeecfb Output: a=2 b=2 c=2 d=1 e=2 f=1"

Ans.
let str="abcadeecfb";

let mp=new Map();
for(let i=0;i < str.length;i++){
  let ch=str.charAt(i)
  if(mp.has(ch)){
    let value =mp.get(ch);
    mp.set(ch,value+1)
  }
  else{
    mp.set(ch,1)
  }

}
console.log(mp)
